# 美联储的高利率，正在悄悄分化美国普通人的钱包

上周末我去了一趟东湾的 Costco，顺便和几个开卡车搞长途物流的兄弟聊了聊。我发现，高利率对美国社会的渗透，比华尔街那些冷冰冰的非农数据要残酷得多。

你在新闻里看到美国零售数据还挺好，但你到超市门口看看，大家的购物车里装的都是什么。
以前推车里全是各种有机中高端牛肉、红酒；现在呢？推车里基本是性价比最高的散装鸡肉、面包和打折的家庭装生活必需品。
这就叫“消费降级”，而且是润物细无声的降级。

更深的一层是，美国现在是明显的“两个世界”。
如果你是 2021 年前买了房，锁定了 3% 以下的三十年期固定贷款利率，那你现在的日子过得还挺滋润，因为你最大的刚性支出被锁死了。
但如果你是这两年刚毕业、想在湾区租房或者买房的年轻人，那就真的被高利率“爆锤”了。7% 的房贷利率，能直接把你每个月的可支配现金流腰斩。

这就直接导致了一线就业市场的体感降温。卡车司机老李跟我抱怨，现在全美柴油价格虽然稳住了，但“货却变少了”。
以前码头卸完货，运力抢着要；现在是卡车在港口外面排队，等一单货要多花两个小时。这说明底层的物理供应链，其实已经在主动去库存、收缩战线了。

所以，别只听鲍威尔在发布会上说什么“美国经济极具韧性”。
真正决定流动性走向的，是这些在 Costco 门口精打细算的普通家庭，和在港口排队等货的卡车司机。他们的口袋，才是美联储降息的终极发令枪。
