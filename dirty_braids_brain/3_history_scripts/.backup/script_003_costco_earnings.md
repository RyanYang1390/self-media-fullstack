# Costco 财报里藏着的底层密码：为什么它能让中产一边哭一边掏钱？

Costco 刚发了财报，营收和利润都稳稳超过了华尔街的预期。很多人在问：通胀这么高，为什么它的护城河还能这么宽？

我只聊两个我自己的观察。

第一，Costco 的本质根本不是一家零售商，它是一家“会员制俱乐部”。
你看财报会发现，它几乎所有的营业利润都来自它的会员费，而货物的毛利率被压到了极其变态的 11% 左右。这是什么概念？普通超市的毛利率要在 25% 到 30%。
说句人话，Costco 根本不靠卖牛奶和牛肉赚钱。它是把货物的进价加上极低的运营损耗，原封不动地交到你手里，它只收你的“门票钱”。
这就是巴菲特说的“无形资产和超强的转换成本”。你一旦交了 120 美金的年费，你就会产生一种“我必须多去几次把本赚回来”的心理。这就直接锁定了你的消费频次。

第二，是它极致的商业模式：做减法。
普通的沃尔玛或者 Target，里面有几万个 SKU（品类）；但 Costco 每一个品类通常只提供 2-3 个最头部的品牌，总 SKU 只有 4000 个左右。
这个细节大家很容易忽略。因为品类少，所以它对单一供应商的采购量是天文数字。
这就直接拿到了全行业最低的批发价，形成了碾压级别的“成本优势”。它把这个省下来的成本再补贴给消费者，消费者粘性就更强，这就形成了一个牢不可破的“飞轮效应”。

上周我跟一位德州做食品分销的兄弟聊。他说，只要你能把货送进 Costco 的供应链，你的厂就能活得极度滋润。但是它对品质和价格的卡控，也极其严苛。

在这个全球流动性偏紧、中产钱包被挤压的节骨眼上，谁能帮大家省钱，谁就是真正的赢家。Costco 的护城河，在这个周期里不仅没有变窄，反而被它的会员卡越挖越深了。
