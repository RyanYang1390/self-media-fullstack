# 美股狂飙，我却在Costco看到了消费冰点：ETH/BTC逃顶路线图

昨天我去了趟 Palo Alto 的牛排馆，周中晚上空无一人。这比华尔街任何好看的非农数据都让我发冷。

今天我们只聊一个大家非常关心的事：美股大盘在历史高点狂飙，标普500都创了新高，席勒市盈率直逼 2000 年互联网泡沫。但同时，密歇根大学的消费者信心指数，却跌到了 70 年来的历史最低点。

这正常吗？股市和消费，为什么会割裂得这么严重？

我只说一个核心逻辑：**现在的股市繁荣，其实是一个“生产端”的狂欢，跟“消费端”已经彻底脱钩了。**

你看到 NVIDIA 它们财报利润飙升，市值狂涨，那是因为这些公司在拼命搞“降本增效”。AI 确实能帮你砍掉成本，提升效率，短期内财报肯定好看。股市在为这种“降本增效”的叙事狂欢。

但问题的另一面是，当这些大厂拼命“降本”的时候，很多普通人就成了被降的“本”。高昂的汽油、生活成本，加上 4.4% 的核心通胀，正在实质性地摧毁底层消费者的购买力。

说句人话，AI 确实能帮你把广告投得更精准。但如果大家都被“优化”掉了，口袋里没钱，连基本生活都难以为继，那你的“精准广告”再有效，谁来买单？

这就是一个“死循环”：
消费端崩溃 ➔ 广告总盘子萎缩 ➔ 广告主赚不到钱，肯定砍投放 ➔ Meta、Google 营收就下来了 ➔ 他们就没钱再买 NVIDIA 的芯片和算力了 ➔ 整个 AI 产业链的发动机，就熄火了。
这直接会导致整个市场的估值全面坍塌。

那为什么现在还没崩？这里有三个“摩擦力变量”在拉长这个过程：
第一，**时间错配**。大厂裁员降本，利润立竿见影。但消费坍塌，是一个缓慢失血的过程，像信用卡刷爆、失业传导，它有滞后性。
第二，**囚徒困境**。科技巨头现在买芯片，不是为了赚即时广告费，而是为了“活下去”。在“底池套牢”的状态下，他们即使借债也必须继续买芯片，这拉长了后端的缓冲期。
第三，**马太效应**。在需求萎缩初期，广告主会追求极端确定性，砍掉传统媒体，把余粮都集中投给 Meta、Google 这些“转化率之王”。这就造成了巨头短期内市场份额扩大的虚假繁荣。

所以，我们现在就处在这个传导链条的：**第一阶段尾声，正在向第二阶段过渡。**
上周沃尔玛的财报已经露了底牌：连中高收入家庭，都在拼命往沃尔玛这种“平价商超”跑，说明连他们的“消费垫”都快被通胀击穿了。大众消费端，隐患已经积累到极致了。

我们普通人，该看什么指标来提前避险？
第一，看**纽约联储的信用卡/车贷违约率**。如果 30/90 天逾期率开始倾斜上升，超越 2008 年或 2019 年前高，那就是底层消费断裂的信号。
第二，看**四大科技巨头（Hyperscalers）的 CapEx 指引**。如果管理层在财报电话会上，首次流露出“由于宏观不确定性，要放缓或优化算力中心建设”的信号，那就是全面撤退的红灯。

那我们手里的 Web3 资产，ETH 和 BTC，又该怎么看？
**别把 ETH 当成数字黄金。** 它更像高贝塔值的科技股，是链上经济的“汽油”。科技股一崩，链上活动就熄火，TVL 暴跌。对冲基金还会把它当成流动性最好的筹码，无差别抛售去补保证金。
**BTC 呢？** 它更像“数字黄金”。在 4.4% 的微观滞胀环境里，它具备更强的主权避险和抗通胀属性，明显比 ETH 抗跌。

所以，我的三阶段博弈策略是：
**第一阶段（当前风暴前夕）**：主抱 **BTC + 稳定币现金**，对 ETH 保持高度克制。高利率对利息资产和抗通胀的 BTC 有利，ETH 链上活动受限。
**第二阶段（雪崩爆雷发生时）**：**Cash is King！** 所有风险资产短期期望值都是负的。持有充裕法币或顶级稳定币，保留子弹，冷眼等待废墟。
**第三阶段（风暴后大印钞启动）**：当美联储被迫降息放水，印钞机轰鸣的时候，法币大幅贬值。此时，那些被无差别错杀、具备强生产力属性的 **ETH**，将迎来盈亏比惊人的“黄金坑”左侧建仓机会！

在这个周期里，看懂底层逻辑，握紧手里的现金流，是咱们普通人拿住的唯一底牌。下期聊。
